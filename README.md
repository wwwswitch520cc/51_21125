<p><strong>年轻的瓦西丽莎注定要成为一名女巫，但她决定无视自己的命运并嫁给自己的心上人。不过随着她未婚夫的离奇死亡，她的这个梦想也破灭了。</strong></p>
<p><strong>失去爱人的瓦西丽莎心如刀绞，于是踏上了寻找黑色之书的旅途。传说这件恶魔神器强大无比，只要解开它的 7 个封印，它就能满足一切愿望。</strong></p>
